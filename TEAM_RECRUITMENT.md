## 详细职位描述

### 1. PTP Lead Engineer (1.0 FTE)

**核心职责**
- 领导 PTP 协议栈从 0 到 1 的开发 (9 个月)
- 架构设计与代码审核
- 与硬件团队协作实现跨平台时间戳精度验证
- 技术文档编写与开发者培训

**必需技能**
- ✅ 5+ 年 C/C++ 开发经验 (Linux/Embedded)
- ✅ 深入理解网络协议 (TCP/IP, UDP, VLAN)
- ✅ 有精密时间同步经验 (优选: NTP/PTP/GPS)
- ✅ 熟悉多平台开发 (至少 2 个以上: STM32, Linux, RTOS)

**加分项**
- ⭐ IEEE 1588 或 NTP 实现经验
- ⭐ 发表过网络/时间同步相关论文
- ⭐ 开源项目维护者
- ⭐ 机器人领域背景

**期望背景**
- 国内 Top 500 技术企业 (华为、大疆、Tesla 等)
- 或知名芯片设计公司 (ARM, Qualcomm, Intel)
- 或学术背景 (清华、北大、浙大 博士级别)

**招聘渠道优先级**
1. **LinkedIn** - 直接向 10+ 候选人发送 InMail
   - 搜索关键词: "NTP" OR "time synchronization" OR "IEEE 1588"
   - 地区: 北京、深圳、杭州
   - 经历: 5-10 年

2. **CSDN & 开源社区**
   - 搜索 Linux 时间系统相关项目贡献者
   - 联系 OpenHarmony / Linux kernel 活跃贡献者

3. **大学校友网络**
   - 清华 EE/CS 系校友推荐
   - 业界人脉转介

4. **猎头推荐**
   - 联系专业猎头 (HR market rate: 25% commission)
   - 目标: 2 周内推荐 5+ 候选人

**面试流程** (共 4 轮, 2 周完成)
```
第1轮: 技术笔试 (2小时)
├─ C 代码题 (2个): 内存管理 + 数据结构
├─ 系统设计题 (1个): 时间同步架构设计
└─ 评分: >70% 进入第2轮

第2轮: 系统设计面试 (1.5小时)
├─ 题目: "如何设计一个 <10µs 的时间同步系统?"
├─ 考察: 架构、权衡、性能优化
└─ 评分: 通过进入第3轮

第3轮: 技术深度面试 (1小时)
├─ 现有项目经验讲解 (20分钟)
├─ 深度技术问题 (35分钟)
│  ├─ IEEE 1588 state machine
│  ├─ PI servo 算法
│  └─ 硬件时间戳精度
└─ 评分: 通过进入第4轮

第4轮: CEO 面试 (45分钟)
├─ 价值观与协作方式
├─ 长期职业规划
└─ 股票期权与薪资谈判
```

**薪资与福利**
- 基本薪资: 35-50K 元/月 (根据经验)
- 绩效奖金: 年底 0-5 月 (取决于里程碑达成)
- 期权: 0.5-1.0% (4 年 cliff, 1 年 vesting)
- 健康保险: 五险一金 + 商业保险
- 福利: 远程工作弹性, 年假 15 天

### 2. TSN Network Engineer - Senior (1.5 FTE)

**核心职责**
- TSN 网络调度系统设计与实现
- 硬件交换机集成与性能验证
- Linux kernel taprio / qdisc 优化
- 性能基准测试与文档

**必需技能**
- ✅ 4+ 年 Linux 内核与网络编程经验
- ✅ IEEE 802.1 标准深入了解 (Qbv, Qci, SRP)
- ✅ 硬件网络交换机配置经验
- ✅ 性能基准测试与瓶颈分析

**加分项**
- ⭐ Linux kernel networking subsystem 代码贡献
- ⭐ TSN 相关实现经验 (如 TTTech AutoCore)
- ⭐ 车联网/工业物联网背景
- ⭐ 论文发表 (QoS, network scheduling)

**期望背景**
- 国内外网络设备厂商 (华为、思科、Marvell 等)
- 或知名互联网公司的网络基础设施团队
- 或学术机构 (CCF 论文发表者优先)

**招聘渠道**
1. **技术社区**
   - Linux 内核 TC (Traffic Control) subsystem contributors
   - GitHub 搜索: "taprio" OR "IEEE 802.1" 高 star 项目

2. **行业论坛**
   - Linux Foundation 会员名单
   - 开源网络峰会参会者

3. **高校合作**
   - 清华、北邮通信学院教授推荐
   - 国防科大网络安全相关研究生

**面试重点**
- Linux 内核代码阅读能力
- TSN 标准理解 (给出论文, 讨论实现)
- 性能优化的系统思维
